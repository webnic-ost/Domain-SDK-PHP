Webnic\WebnicSDK\Services\DNS\DNSSubscription::addDomainZoneToPartnerSubscription ( string $zone)

Adds a domain zone to a partner subscription.

This endpoint adds a domain zone to a partner subscription. Zone usage will be consumed. This endpoint is not applicable for non-WebNIC domains.

Parameters
string$zoneThe domain zone to add to the partner subscription.
Returns
array The API response, including:
  • string zone: The domain zone name.
  • string zoneType: The zone type (Possible values: "inzone" for WebNIC domain, "outzone" for Non-WebNIC domain).
  • string subscription: The partner subscription type.
  • string subscriptionId: The partner subscription ID.
  • string dtcreate: The date and time when the domain zone was created.
  • string dtmodify: The date and time when the domain zone was last modified.

Example request:

$zone = "example.com";
$response = $webnicSDK->DNSSubscription->addDomainZoneToPartnerSubscription($zone);

Webnic\WebnicSDK\Services\DNS\DNSSubscription::getDomainSubscription ( array $filters)

Retrieves domain subscriptions with optional filtering criteria.

Parameters
array$paramsQuery parameters for filtering the results:
  • domainName (string, optional): Domain name to filter by.
  • subscriptionId (string, optional): Subscription ID to filter by.
  • subscriptionProduct (string, optional): Subscription Product Type. Allowed value: "premium_ns_georoute".
  • subscriptionAutoRenew (string, optional): Indicates if the subscription is set to auto-renew.
  • limit (int, optional): Number of domain subscriptions to be shown. Default: 10, Maximum: 100.
Returns
array The response containing domain subscription details.
Exceptions

Exception If the request fails.
